# プライバシーポリシー

RadarChart（以下、「本アプリ」といいます。）は、本アプリが提供するサービス（以下、「本サービス」といいます。）における、ユーザーの情報の取扱いについて、以下のとおりプライバシーポリシー（以下、「本ポリシー」といいます。）を定めます。

## 1. 取得する情報および利用目的
本アプリでは、提供するサービスの向上やお問い合わせ対応のため、以下の情報を取得する場合があります。

* **お問い合わせ情報**
  * 内容：メールアドレス、お問い合わせ内容
  * 利用目的：ユーザーからのお問い合わせへの回答および本人確認のため
* **利用状況の分析（利用している場合）**
  * 内容：アプリ内での操作ログ、端末識別子
  * 利用目的：サービスの利便性向上および統計データ作成のため

## 2. 第三者提供
本アプリは、法令によって許容される場合を除き、ユーザーの同意を得ることなく、第三者に個人情報を提供することはありません。

## 3. 免責事項
本アプリの利用により生じた何らかのトラブルや損失、損害等に対して、開発者は一切の責任を負わないものとします。

## 4. プライバシーポリシーの変更
本アプリは、法令の制定、改正等により、本ポリシーを適宜見直し、予告なく変更する場合があります。
